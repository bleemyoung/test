# 堆排序 (Heap Sort)

## 基本概念

- **时间复杂度**: O(n log n)
- **空间复杂度**: O(1) 原地排序
- **稳定性**: 不稳定排序
- **数据结构**: 完全二叉树

## 大顶堆性质

每个节点的值 ≥ 其左右子节点的值

```
        13
       /  \
      11   12
     / \   /
    5   6 7
```

## 算法步骤

### 1. 建堆 buildHeap()

从最后一个非叶子节点向上调整

```js
for (let i = Math.floor(n / 2) - 1; i >= 0; i--) {
  heapify(i, n);
}
```

### 2. 堆化 heapify(i, n)

从节点 i 向下调整，维护堆性质

- 比较 i、左子(2i+1)、右子(2i+2)
- 将最大值换到根节点
- 递归调整被交换的子树

### 3. 排序 sort()

```
循环 n-1 次:
  1. 堆顶(最大值)与末尾交换
  2. 对剩余元素堆化
  3. 末尾向前移动
```

## 示例演示

```
原数组:     [12, 11, 13, 5, 6, 7]

建堆后:     [13, 11, 12, 5, 6, 7]

第1轮: 交换13↔7 → [7, 11, 12, 5, 6 | 13]
       堆化       → [12, 11, 7, 5, 6 | 13]

第2轮: 交换12↔6 → [6, 11, 7, 5 | 12, 13]
       堆化       → [11, 6, 7, 5 | 12, 13]

第3轮: 交换11↔5 → [5, 6, 7 | 11, 12, 13]
       堆化       → [7, 6, 5 | 11, 12, 13]

第4轮: 交换7↔5  → [5, 6 | 7, 11, 12, 13]
       堆化       → [6, 5 | 7, 11, 12, 13]

第5轮: 交换6↔5  → [5 | 6, 7, 11, 12, 13]

结果: [5, 6, 7, 11, 12, 13]
```

## 索引关系

| 节点 | 索引 | 左子 | 右子 | 父节点 |
|------|------|------|------|--------|
| 公式 | i | 2i+1 | 2i+2 | (i-1)/2 |

## 方法一览

| 方法 | 作用 | 时间复杂度 |
|------|------|------------|
| constructor | 初始化并建堆 | O(n) |
| buildHeap | 构建大顶堆 | O(n) |
| heapify | 调整堆 | O(log n) |
| sort | 堆排序 | O(n log n) |
