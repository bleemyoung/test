# 力扣题单

这个目录用来按题型整理 LeetCode 复习内容，方便后续按专题回顾，而不是只按题号零散存放。

## 目录说明

- 每个子目录对应一个专题，例如：滑动窗口、并查集
- 专题目录下优先放 `review.md`，用于沉淀该题型的核心套路、易错点和复习入口
- 具体题目的题解仍然可以保留在原题目录中，题单目录侧重“分类导航”和“复习索引”

## 题型分类

- [滑动窗口](./滑动窗口/review.md)
- [并查集](./并查集/review.md)

## 复习建议

- 先看专题 `review.md`，快速回忆这个题型解决什么问题
- 再看该专题下已经整理过的代表题
- 复习时重点关注：识别特征、核心模板、边界处理、常见误区
